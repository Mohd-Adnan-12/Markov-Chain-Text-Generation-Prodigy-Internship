# Markov Chain Text Generation  
### Generative AI Internship – Prodigy InfoTech (Feb 2026)

This project implements a probabilistic text generation system using Markov Chains.

The model learns patterns from user-provided text and generates new sentences based on learned word probabilities.

---

## 📌 Project Objective

To understand sequence prediction and fundamental language modeling using probabilistic methods.

---

## 🚀 Features

- User-input training text
- Probabilistic sentence generation
- Simple Markov Chain implementation

---

## 🛠 Technologies Used

- Python
- Markovify

---

## ▶️ How to Run

pip install -r requirements.txt  
python app.py
